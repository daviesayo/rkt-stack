[2026-05-08 16:00] | promptsmith's `/promptsmith` Stage 2 defaults to interactive (ask via `AskUserQuestion`, wait); ship-with-flags fallback fires only on explicit user skip or async/batch use | Verification testing in async mode never exercised the interactive question path because the spec was ambiguous: it said both "show questions and draft, user reacts" (interactive) and "do not stall on missing input" (one-shot).
